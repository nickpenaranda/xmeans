Notes

  • USE THE DEVELOPER CONSOLE
  • Use `xmeans` object to generate test data and perform clustering
  • xmeans.genTestData(dims, n) generates n uniformly random dims-dimensional points
  • xmeans.genTestData2(dims, n, k, sd) generates n, dims-dimensional points clustered around k centroids with standard deviation of sd
  • xmeans.genTestData3() generates 2d clustered data on a diagonal
  • xmeans.xmeans(data, options) returns clustering data. options is a plain object with optional properties "kMin" and "kMax", by default 2 and n/4, respectively.
  • xmeans.oxmeans(data, options) runs xmeans.xmeans(...) repeatedly looking for the best model (maximize BIC). options is a plain object that accepts optional properties for xmeans.xmeans(...) and an additional option, "iters" (default 3).
  • xmeans.render(clusters) to render the output of xmeans.xmeans(...)
